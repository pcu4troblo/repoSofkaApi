Pasos de Ejecucion
1. Descargar e instalar JDK 11 con variables de entorno
2. Descargar e instalar GRADLE 7.6.1 con variables de entorno
3. Dirigirse a la ruta del proyecto en linea de comandos
4. Ejecutar el siguiente comando: gradle :test --tests "runner.DemoblazeRunner"
5. Ingresar al reporte generado después de cada ejecución, en la ruta "target/cucumber-html-reports/overview-features.html"
